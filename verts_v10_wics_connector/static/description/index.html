API Authentication with Odoo Queue job

Add Some Congiguration in configuration file

1) Add this in Config file :
[options]
; This is the password that allows database operations:
admin_passwd = admin
db_host = localhost
db_port = 5432
db_user = user
db_password = password
addons_path = /home/nihal/workspace/odoo-10.0/addons
xmlrpc_port = 1070
workers = 4
server_wide_modules = web,queue_job
[queue_job]
channels = root:1
root.connector_jira.import:

Run these 4 commands from ur terminal --
(1) export PATH=ODOO_QUEUE_JOB_HTTP_AUTH_USER=admin
(2) export PATH=ODOO_QUEUE_JOB_HTTP_AUTH_PASSWORD=adminpassword
(3) export PATH=ODOO_QUEUE_JOB_JOBRUNNER_DB_HOST=localhost
(4) export PATH=ODOO_QUEUE_JOB_JOBRUNNER_DB_PORT=5432
Note : - adminpassword is whatever admin password you have to set on your db
and then run server
and use always dbfilter on cammand like python ./odoo-bin -c o10.conf --db-filter=db_name
